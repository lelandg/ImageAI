## About ImageAI

ImageAI is a cross-platform desktop application (Windows, macOS, Linux) built with Python and PySide6. It supports both a graphical interface and a command-line interface.

Supported image generation providers: Google Gemini, OpenAI DALL-E, Stability AI, Stable Diffusion (local), Midjourney, and Ollama.

Current version: 0.32.0
