# Layout & Books

## Overview

The Layout/Books tab is a professional layout engine for designing photo books, children's books, comic books, and magazines. Combine AI-generated images with your own text to create publication-ready pages you can export as PDF or PNG.

## Features

### Template Selection

Start from a built-in template suited to your project type:

| Template | Best for |
|----------|---------|
| Children's Book | Large images, simple text blocks |
| Comic Book | Panel grids, speech bubbles, action layouts |
| Magazine | Multi-column text, photo captions |
| Photo Book | Full-bleed images, minimal text |

Select a template when creating a new layout. You can switch templates mid-project — your content is preserved and re-flowed into the new design.

### Working with Images

Place any image into a layout cell or panel:

- Import AI-generated images directly from your ImageAI history.
- Import images from your computer (PNG, JPEG, WebP).
- Apply per-image styling:

| Effect | Options |
|--------|---------|
| Corners | Rounded (adjustable radius) or square |
| Filters | Blur, Grayscale, Sepia, Sharpen |
| Borders | Solid, adjustable thickness and color |

Images are scaled proportionally to fit their panel. Select "Fill" to fill the panel (cropping edges) or "Fit" to show the whole image with letterboxing.

### Typography and Text Layout

Add text blocks to any panel or page area:

- **Word wrapping** — text wraps automatically to the column width.
- **Hyphenation** — long words are hyphenated at natural break points.
- **Widow/orphan control** — prevents single words or lines from being stranded at the top or bottom of a column.
- **Justification** — left, right, center, or full justify.
- **Font selection** — choose from system fonts or add custom fonts to the Fonts directory.

### Font Generator Integration

Create custom fonts from hand-drawn or AI-generated alphabet images and use them in your layouts. See the Font Generator guide for details.

### Page Management

- Add, duplicate, and delete pages from the page navigator.
- Drag-and-drop to reorder pages.
- Preview all pages in the thumbnail panel before export.

### Export Options

#### PDF Export

- Embedded fonts — fonts are baked into the PDF so it looks correct on any device.
- 300 DPI print resolution for professional print quality.
- Single-file output containing all pages.

#### PNG Sequence

- One PNG file per page.
- 300 DPI resolution.
- Suitable for uploading individual pages to print-on-demand services.

## Common Questions

**Q: Can I mix portrait and landscape pages in one project?**
No — all pages in a project share the same page size and orientation. Start a new project for a different orientation.

**Q: How do I add a speech bubble for a comic panel?**
Use a text block with a rounded rectangle border applied. Full speech bubble shapes with tails are on the roadmap.

**Q: Can I use images from outside ImageAI in my layout?**
Yes — click Import Image in any panel to browse for any PNG or JPEG on your computer.

**Q: The PDF looks different from the screen preview. Why?**
Make sure all fonts used in the layout are available on your system. If a font is missing at export time, it falls back to a system default. Use fonts from the Fonts directory for consistent results.

**Q: What is the maximum page count?**
There is no enforced limit, but very large projects (50+ pages with high-resolution images) may use significant memory. Save frequently on large projects.
