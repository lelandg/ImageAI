# Site Navigation

> A guide to every major section of ImageAI and how to find what you need.

## Application Tabs

ImageAI is organized into tabs along the top of the main window.

### Generate (`Generate` tab)

The main creative workspace. Write a prompt, choose a provider and model, set your aspect ratio, and click Generate.

Key sections within the tab:
- Prompt field — multi-line text entry with Ctrl+F search
- Provider / Model dropdowns — select your AI provider and model
- Aspect ratio buttons — 1:1, 3:4, 4:3, 16:9, 9:16, 21:9, or custom
- Reference Images panel — add reference photos to guide generation
- AI Prompt Tools — Enhance Prompt, Generate Prompts, Ask AI Anything
- Advanced Controls — inference steps, CFG, seed, negative prompt (provider-dependent)
- Cost estimate — shown before each generation
- Generated image preview — appears on the right side after generation

### Prompt Builder (`Prompt Builder` tab)

Structured prompt creation by clicking categories. Includes presets and export/import.

### Video Projects (`Video` tab)

AI video creation from text scripts or lyrics. Includes the scene storyboard, MIDI sync, karaoke overlays, and rendering controls.

### Layout / Books (`Layout` tab)

Page layout editor for photo books, comics, and publications. Templates, typography controls, image styling, and PDF/PNG export.

### Character Animator (`Character` tab)

Guided wizard to convert a character image into an Adobe Character Animator puppet.

### Font Generator (`Fonts` tab)

Wizard to convert an alphabet image into a TTF or OTF font file.

### History (`History` tab)

Full log of every generated image with thumbnails, metadata, search, and sorting. Double-click any row to reload all settings.

### Settings (`Settings` tab)

Configure API keys, authentication, output directory, image format, Discord rich presence, and upscaling.

### Help (`Help` tab)

In-app documentation with full-text search (F3 / Shift+F3).

## Quick Navigation Reference

| What You Want to Do | Go To |
|--------------------|-------|
| Generate an image | Generate tab |
| Build a prompt step-by-step | Prompt Builder tab |
| Enhance or rewrite a prompt | Generate tab → Enhance Prompt button |
| Use a reference photo | Generate tab → Reference Images panel |
| Find a reference image on Wikimedia | Generate tab → Search Wikimedia button |
| Upscale an image | Settings tab → Upscaling section |
| Create a music video | Video tab |
| Add karaoke overlays | Video tab → Karaoke section |
| Make a photo book or comic | Layout tab |
| Create a puppet for animation | Character tab |
| Make a font from an image | Fonts tab |
| Find a past generation | History tab |
| Change API key or provider | Settings tab |
| Change output folder | Settings tab → Output Directory |
| Toggle Discord Rich Presence | Settings tab → Discord Rich Presence |
| Read documentation | Help tab |
