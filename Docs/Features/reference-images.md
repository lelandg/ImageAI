# Reference Images

## Overview

Reference images let you guide AI generation using photos or artwork you already have. Instead of describing everything in text, you show ImageAI what you want and it uses those visuals as creative input.

## Features

### Three Reference Modes

#### Flexible Mode (Google Gemini)

Best for style transformation and character design. You can add as many reference images as you like.

- Upload photos and the AI applies the artistic style from your references.
- When you add two or more images, ImageAI automatically composites them into a character design sheet — useful for turning multiple views of a character into a single reference grid.
- Ideal for converting photos to cartoons, anime, or painted styles.

How to use:
1. Open the Reference Images panel in the Generate tab.
2. Select Flexible mode.
3. Click Add Image and pick one or more files (PNG, JPEG, or WebP).
4. Write your prompt as usual and click Generate.

#### Strict Mode (Imagen 3 Customization)

Best for maintaining precise identity consistency — great for products, characters, or branded assets.

- Add up to 3 reference images.
- For each image, choose a reference type:

| Type | When to Use |
|------|-------------|
| SUBJECT | Person, animal, or specific product |
| STYLE | Artistic style you want to replicate |
| CONTROL | Structural guide (canny edge, scribble, face mesh) |

- Optionally add a description and a reference ID ([1], [2], or [3]).
- Use the reference ID in your prompt to call out a specific reference: "make the character in [1] pose like [2]".

#### Edit Mode

Start from an image you already generated and refine it.

1. Double-click any image in History, or click Load Image in the Reference panel.
2. The original prompt, settings, and metadata load automatically.
3. Adjust your prompt and click Generate to create a refined version.
4. The source image is preserved — Edit Mode creates a new image alongside the original.

### Supported Input Formats

- PNG
- JPEG / JPG
- WebP

### Wikimedia Commons Search

If you need a reference image but do not have one ready, use the built-in Wikimedia search:

1. Click Search Wikimedia in the Reference panel.
2. Enter keywords.
3. Filter results by license (Public Domain, CC BY, CC BY-SA).
4. Click an image to preview it, then Download to add it directly to the reference panel.

All downloaded images include their attribution and license information.

## Common Questions

**Q: How many reference images can I use in Flexible mode?**
There is no hard limit — you can add as many as you like. More references give the AI more style information to work from.

**Q: In Strict mode, what happens if I add more than 3 images?**
The panel will only allow 3 images in Strict mode. Use Flexible mode if you need more.

**Q: Can I mix Flexible and Strict modes?**
No — you select one mode per generation. Switch the mode selector before generating.

**Q: I loaded an image into Edit mode but the prompt is blank. Why?**
Edit mode loads metadata from ImageAI's JSON sidecar files. If the image was not originally generated by ImageAI, no metadata file exists and the prompt will be empty.

**Q: What is canny/scribble/face_mesh under CONTROL type?**
These are structural control maps. Canny extracts edges from your reference, scribble uses rough drawn lines, and face_mesh maps facial landmarks. They tell the AI how to position or shape the output.
