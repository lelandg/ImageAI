Complete reference for favicon dimensions across platforms and devices.

## Size Recommendations by Use Case

### Minimum Set (2025)
For basic favicon support across all browsers:
- **32×32** - Standard modern favicon
- **180×180** - iOS Apple Touch Icon

### Recommended Set (2025)
For comprehensive support including high-DPI displays:
- **32×32** - Standard favicon
- **48×48** - Chrome/Google
- **96×96** - 2x retina
- **180×180** - iOS
- **192×192** - Android

### Complete Set (2025)
For maximum compatibility and quality across all devices:
- **16×16** - Legacy support
- **32×32** - Standard
- **48×48** - Chrome
- **96×96** - 2x retina
- **144×144** - 3x retina
- **180×180** - iOS
- **192×192** - Android

## Technical Notes

- **File Format**: PNG offers best compatibility; SVG is supported by modern browsers
- **Optimization**: Favicons should be under 15KB for fast loading
- **DPI**: All sizes are at 72-96 DPI (standard screen resolution)
- **Multiples of 48**: Wix and some platforms prefer sizes that are multiples of 48px (48, 96, 144, 192)
- **Google Search**: Chrome displays favicons in search results at 48×48, so this size is important for SEO
