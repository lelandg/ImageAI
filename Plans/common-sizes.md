## Size Recommendations by Use Case

### Web Images
- **1280-1920px width**: Optimal range for web images
- **Under 200KB**: Target file size for fast loading
- **WebP format**: Offers 25-30% smaller files than JPEG
- **Responsive**: Consider multiple sizes for different screen sizes

### Print Images
- **300 DPI**: Standard for high-quality prints
- **CMYK color**: Convert from RGB for professional printing
- **Bleed area**: Add 0.125" (37px @ 300 DPI) for trim
- **TIF/PNG format**: Lossless formats for print

### HD Video
- **16:9 aspect ratio**: Universal video standard
- **1920×1080**: Minimum for "Full HD" designation
- **4K (3840×2160)**: Future-proof for high-resolution displays
- **60 FPS**: Smooth motion for video thumbnails/covers

### Social Media
- See `social-media-image-sizes-2025.md` for platform-specific requirements

## Technical Notes

- **Web DPI**: 72-96 DPI is standard for screen display
- **Print DPI**: 300 DPI minimum for professional printing, 600+ for fine art
- **File Format**:
  - Web: WebP, JPEG, PNG
  - Print: TIFF, PNG (lossless)
  - Transparency: PNG, WebP
- **Optimization**: Always compress images for web use
- **Scaling**: Scale down from higher resolution, never up

## Print Size Calculator

To calculate print size in pixels:
- **Formula**: `pixels = inches × DPI`
- **Example**: 8×10 photo @ 300 DPI = (8×300) × (10×300) = 2400×3000px
